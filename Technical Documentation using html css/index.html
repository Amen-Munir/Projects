C++ Entry Point

The entry point to C++ application is the main function

The main function is the start of the program in c++. ontrol Shifts to the main function as sson as you press run and executes all the code line by line. 

          
            #include

            }

As you can see the main function is declared with the return type of integer. And the paramter list is empty. The main funtion can also e written with a return type of void but the normal practice is to use it with the return type of integer.

Printing to the cosole

In order to prit to the console, we can use the cout statements as shown in the above code as well.

The cout statement is used to print to the console. It is a part of the iostream library. The cout statement is used with the insertion operator (<<) and the text to be printed is enclosed in double quotes. 

                  
              int main(){
                cout << "My first line of code";
                return 0;
              }

In this example we are printing the string "My first line of code" to the console when we run the program.

The cout statement can also be used to print variables to the console. Anything inside the double quotes will be printed as it is whereas to print variables to the console you simply write in the variable name.

C++ Comments

Comments can be used to explain C++ code, and to make it more readable. It can also be used to prevent execution when testing alternative code. Comments can be singled-lined or multi-lined.d

Singled-lined Comments

Single line comments start with two forward slashes (//). Any text between // and the end of the line is ignored by the compiler (will not be executed).

            Example:
          
            // This is a comment
            cout << "Hello World";

Multiple-lined Comments

Multi-line comments start with /* and ends with */. Any text between /* and */ will be ignored by the compiler:.

            Example:
          
            /* The code below will print the words Hello World!
            to the screen, and it is amazing */
            cout << "Hello World!";
Data Types in C++

In C++, there are different types of variables (defined with different keywords), for example:



  • int - stores integers (whole numbers), without decimals, such as 123 or -123
  • double - stores floating point numbers, with decimals, such as 19.99 or -19.99
  • char - stores single characters, such as 'a' or 'B'. Char values are surrounded by single quotes
  • string - stores text, such as "Hello World". String values are surrounded by double quotes
  • bool - stores values with two states: true or false





Some copiable code: 

                
                  int main(){
                    int a = 5;
                    double b = 5.5;
                    char c = 'A';
                    string d = "Hello World";
                    bool e = true;
                  }

C++ Variables

Declaring (Creating) Variables



To create a variable, specify the type and assign it a value: 

          
Syntax:

          
            type variable = value;

Where type is one of C++ types (such as int), and variableName is the name of the variable (such as x or myName). The equal sign is used to assign values to the variable.


To create a variable that should store a number, look at the following example:

          
Syntax:

          
            int myNum = 15;
            cout << myNum;


Note that if you assign a new value to an existing variable, it will overwrite the previous value:

          
Syntax:

          
            int myNum = 15;  // myNum is 15
            myNum = 10;  // Now myNum is 10
            cout << myNum;  // Outputs 10

Object Oriented Programming

C++ is known as an Object Oriented programming language.

This means that it is easy to represent real life objects in code by using classes and encapsulation.

An example of this might be a Student class that has a name and age property. 

          
            public class Student{
                //Student properties
                private :
                string name;
                 int age;

                //Constructor
                public :
                Student(String n, int a){
                   name=n;
                   age=a;
                }
                
                //Getter method 
                 string getName(){
                    return name;
                }
                 
                 int getAge(){
                    return age;
                }

                //setter method
                void setName(String n){
                    name = n;
                }
            }

We use this class by doing the following: 

            
                Student student = new Student("John", 20);
                student.setName("Jane");
                string name = student.getName();
